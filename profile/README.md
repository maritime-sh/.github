### Why Maritime?

Traditional cloud wasn't built for AI agents. Serverless functions time out. Containers waste money sitting idle. VMs require DevOps expertise. Maritime changes this with a purpose-built runtime that understands how agents actually work.

**Sleep/Wake Architecture** &mdash; Agents automatically sleep when idle and wake on-demand in under a second. You only pay for actual compute, not idle time.

**One-Click Deploy** &mdash; Connect your GitHub repo. We handle the build, deploy, SSL, and routing. No Dockerfiles, no YAML, no CI/CD pipelines to maintain.

**Built-in Triggers** &mdash; Invoke agents via cron schedules, webhooks, REST API, or messaging integrations. Everything you need to orchestrate agent workflows, out of the box.

---

### How It Works

```
Connect repo  →  One-click deploy  →  Agent goes live  →  Sleeps when idle  →  Wakes on trigger
```

Every agent gets a stable HTTPS endpoint, real-time log streaming, environment secret management, and a full lifecycle API. Deploy agents built with **CrewAI**, **LangGraph**, **OpenClaw**, or any custom framework.
